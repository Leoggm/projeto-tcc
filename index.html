IMC

O IMC é uma fórmula utilizada por vários profissionais de saúde para saber, de uma forma rápida, se a pessoa está com o peso adequado ou se precisa ganhar ou perder peso.

De acordo com o resultado de IMC também é possível prever o risco que cada pessoa tem de desenvolver doenças crônicas. Isto porque quanto maior é o valor do IMC, maior é a quantidade de gordura acumulada no corpo e maior o risco de doenças como pressão alta, diabetes e problemas cardíacos.

IMC (kg/m²) Classificação
Menor que 18,5 Baixo peso
De 18,5 a 24,9 Peso normal
De 25 a 29,9 Sobrepeso
De 30 a 34,9 Obesidade grau I
De 35 a 39.9 Obesidade grau II
Igual ou maior que 40 Obesidade grau III

Peso Estimado

A avaliação antropométrica é um método utilizado para investigar o estado nutricional de uma pessoa. Um dos métodos utilizados para descobrir o peso de pacientes sem o uso de balanças, é o de peso estimado da Chumlea.

Para se calcular o peso estimado, quando o paciente não tiver condições de deambular, seu peso deve ser obtido por estimativa, utilizando as fórmulas de Chumlea e colaboradores, aplicando as medidas da altura do joelho (AJ) e da circunferência do braço (CB).

Fórmula Chumlea:
18 a 60 anos:
Homem branco = (AJ x 1,19) + (CB x 3,21) - 86,82
Homem negro = (AJ x 1,09) + (CB x 3,14) - 83,72
Mulher branca = (AJ x 1,01) + (CB x 2,81) - 60,04
Mulher negra = (AJ x 1,24) + (CB x 2,97) - 82,48
Idosos (acima de 60 anos):
Homem branco = (AJ x 1,10) + (CB x 3,07) - 75,81
Homem negro = (AJ x 0,44) + (CB x 2,86) - 39,21
Mulher branca = (AJ x 1,09) + (CB x 2,68) - 65,51
Mulher negra = (AJ x 1,50) + (CB x 2,58) - 85,22

AJ = Altura do Joelho em cm
CB = Circunferência do braço em cm

Taxa de Metabolismo Basal

A Taxa de Metabolismo Basal (TMB) é a quantidade mínima de energia que o corpo precisa em repouso para manter as funções vitais. Ela varia de pessoa para pessoa e é influenciada por fatores como idade, sexo, peso e níveis de atividade física.

Para determinar a quantidade de calorias que você deve ingerir para ganhar peso com base na sua Taxa de Metabolismo Basal (TMB), você precisa considerar o seu nível de atividade física diária. A TMB representa apenas as calorias necessárias para as funções vitais do corpo em repouso, enquanto o nível de atividade física influencia nas suas necessidades energéticas totais.

Equação de Harris-Benedict:
Para Homens:
TMB = 88.362 + (13.397 x P) + (4.799 x A) - (5,677 x I)
Para Mulheres:
TMB = 44.593 + (9.247 x P) + (3.098 x A) - (3.330 x I)
Nível de Atividade:
Sedentário: TMB x 1.2
Levemente ativo: TMB x 1.375
Moderadamente ativo: TMB x 1.55
Muito ativo: TMB x 1.725
Extremamente ativo: TMB x 1.9

P = Peso
I = Idade
A = Altura
